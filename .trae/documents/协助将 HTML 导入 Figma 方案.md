### 核心方案：使用 `html.to.design` 插件 + 本地服务器
Figma 原生不支持直接打开 HTML，最成熟的方案是使用 **html.to.design** 插件。为了保证样式（CSS）、图片和脚本（特别是你的项目包含 `sw.js` Service Worker）能完美还原，强烈建议通过 **本地服务器** 地址导入，而不是直接上传文件。

### 执行步骤
1.  **我将执行的操作**：
    *   在当前目录启动一个临时的 Python HTTP 服务器（端口 8000）。
    *   为你整理出所有页面的访问链接列表。
2.  **你需要执行的操作**：
    *   在 Figma 中搜索并运行插件 `html.to.design`。
    *   选择 **"Import via URL"** 模式。
    *   将我提供的链接（如 `http://localhost:8000/index.html`）填入并点击 Import。

### 为什么这样做？
*   **完整性**：直接拖入 HTML 文件往往会丢失引用的图片或 CSS，而通过 `localhost` 访问可以完美解析所有相对路径。
*   **PWA 支持**：你的目录包含 `manifest.json` 和 `sw.js`，这些 PWA 特性必须在 HTTP 环境下才能生效。
